General Overview

This site named 'Ataraxia' is designed to be a online resource to help people with resting the mind. The website is focused on mental health and ways to help people cope and relieve. Advice and activities about each specific topic can be found under the different pages which are located in the navigation bar at the top.

Bonus Marks

A search bar has been included in the header of every page which allows text to be entered into. However, it does not actually function and is not fully responsive.

An external CSS file named style.css has been used to ensure the website has a consistent theme throughout. Styling for images and tables have been made on separate CSS files for each html page.

In the table on the 'Fitness' page, the rows change colour when the mouse is hovered over the row to add more responsiveness to the website.

At the bottom of the 'Fitness' page, there is a link that can take the viewer back to the fitness table at the top to create more responsiveness and save the viewer from having to scroll all the way back up to the top of the page.

The logo for the website was designed on Canva.

The order of the website pages displayed in the navigation bar and footer will change for every page depending on which webpage the viewer is currently on.
